Files JSON

JSON se utiliza para guardar datos.

Normalmente se trata de arrays con dentro objetos.

Un file JSON devuelve una stringa (contenente todos los datos).

Para poderlo utilizar en Javascript, siendo una stringa, primero hay que pasarlo por JSON.parse(), de esta forma vuelve al formato array/objeto.

VER CONSOLA


Ejemplo:

let = students = `[
{
"name": "Tony",
"age": 37,
"height": 180
},
{
"name": "Bob",
"age": 29,
"height": 172
},
{
"name": "Frank",
"age": 21,
"height": 187
}
]`;


console.log(JSON.parse(students));
